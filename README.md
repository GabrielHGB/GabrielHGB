<h3 align="center">Sobre mim</h3>

###

<p align="left">Sou apaixonado por tecnologia e por aprender coisas novas constantemente, sempre buscando evoluir e entender não apenas como desenvolver soluções, mas como elas realmente funcionam por trás dos bastidores.
